# Employee Data Cleaning Project
## Project Overview
This project demonstrates data cleaning using Python and the Pandas library.

## Tools Used
- Python
- Pandas
- Visual Studio Code
- Git
- GitHub

## Files
- Messy_Employee_dataset.csv (Original dataset)
- Employee_dataset_cleaned.csv (Cleaned dataset)
- clean_employee_data.py (Python cleaning script)

## Data Cleaning Steps
- Removed duplicate records
- Removed extra spaces
- Standardized text formatting
- Converted dates
- Handled missing values
- Removed invalid ages
- Removed negative salaries
- Converted emails to lowercase
- Reset row index
- Saved the cleaned dataset
